# Astronomical Images and Spectra

An interactive walk-through Jupyter Notebook that provides an introduction to astronomical data.

## Structure

- `images_and_spectra.ipynb` - walk-through notebook
- `plots/` - example outputs from the notebook

## Setup

Make sure you have the following Python packages installed:
- numpy
- matplotlib.pyplot
- astropy
- astroquery

using `conda install [package name]` or `pip install [package name]`

## Usage

Download the files into a directory on your machine and start going through the Jupyter Notebook!
